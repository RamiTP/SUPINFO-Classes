# How to install and use libtiff ?

1. Download the latest version of the tiff library source code here : ftp://ftp.remotesensing.org/pub/libtiff (you may need an FTP client like FileZilla)
2. Unzip it and open a shell
3. `cd` into the unzip folder
4. Type the following command to install libtiff on your system :
		./configure
		make
		make install
